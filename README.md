📝 NotesKu - Cloud-Based Notes Application
Aplikasi catatan (Notes App) sederhana yang dibangun dengan arsitektur Fullstack JavaScript dan dideploy menggunakan infrastruktur Google Cloud Platform (GCP). Projek ini dibuat untuk memenuhi tugas mata kuliah Komputasi Awan.

🚀 Fitur Utama
CRUD Operations: Tambah, Lihat, Edit, dan Hapus catatan secara real-time.

Cloud Deployment: Hosting menggunakan Google Compute Engine (VM Instance).

Database Migration: Menggunakan MySQL server yang terkonfigurasi di dalam Cloud VM.

Modern UI: Antarmuka pengguna dengan tema Dark Mode yang responsif.

🛠️ Arsitektur Teknologi
Projek ini menggunakan kombinasi teknologi berikut:

Frontend: HTML5, CSS3, Vanilla JavaScript (Fetch API).

Backend: Node.js & Express.js.

Database: MySQL Server.

Web Server: Apache2 (sebagai penampil file statis).

Infrastructure: Google Cloud Platform (Ubuntu 22.04 LTS).

📋 Prasyarat Sistem
Jika ingin menjalankan projek ini secara lokal, pastikan Anda memiliki:

Node.js (v14 atau lebih baru)

MySQL Server

NPM (Node Package Manager)
